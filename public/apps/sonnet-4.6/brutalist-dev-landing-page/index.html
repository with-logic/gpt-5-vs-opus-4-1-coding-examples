01

Instant Deploy

Push to git, ship to production. Zero config. Zero downtime. Zero surprises.

COPIED! import brutal from '@brutal/sdk'; const api = brutal.deploy({ project: 'my-api', region: 'auto', scale: { min: 0, max: 100 } }); await api.push(); // done.
COPIED! import brutal api = brutal.deploy( project="my-api", region="auto", scale={"min": 0, "max": 100} ) api.push() # done.
COPIED! curl -X POST \ https://api.brutal.dev/deploy \ -H "Authorization: Bearer $KEY"\ -d '{"project":"my-api", "region":"auto"}'
02

Edge Routing

Intelligent traffic distribution across 180+ PoPs. Sub-millisecond routing decisions.

COPIED! const router = brutal.edge({ rules: [ { match: '/api/*', target: 'prod' }, { match: '/beta/*', target: 'canary', weight: 0.1 } ] }); export default router;
COPIED! router = brutal.edge(rules=[ {"match": "/api/*", "target": "prod"}, {"match": "/beta/*", "target": "canary", "weight": 0.1} ])
COPIED! curl -X PUT \ https://api.brutal.dev/routes \ -H "Authorization: Bearer $KEY"\ -d '{"rules":[ {"match":"/api/*","target":"prod"} ]}'
03

Observability

Full-stack tracing, live metrics, structured logs. From request to database, zero blind spots.

COPIED! const span = brutal.trace('checkout'); span.tag({ user, cart_id }); try { await processOrder(cart); span.ok(); } catch (e) { span.error(e); throw e; } finally { span.end(); }
COPIED! with brutal.trace("checkout") as span: span.tag(user=user, cart_id=cart_id) try: process_order(cart) span.ok() except Exception as e: span.error(e) raise
COPIED! curl \ https://api.brutal.dev/traces \ -H "Authorization: Bearer $KEY"\ "?filter=error&since=1h" # returns NDJSON stream
